### Windows Studio 配置注意事项：

- 默认1024*10，如果选配其他模式，有可能Studio没有真正配成功，这个时候需要多配几次，这个是studio的bug
- 如果发现报错，先检查线缆连接与供电，然后用浏览器访问OS1-SN.local或OS-SN.local检查一下是否可以访问，同时Ping一下对应IP或host
